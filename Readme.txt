Steps in creating and deploying files to Github

1. Create local Directory
2. Initialize git init
3. Create files index.js, index.html, inherit.java, angularcli.json & component.ts
4. Create .gitignore file
5. Add angularcli.json & component.ts inside .gitignore file
6. Git add the files - index.js, index.html, inherit.java & .gitignore
7. Git commit the files
8. Create repo in personal github account
9. Add remote for the repo created in Github
10. Push the committed files to the remote
